# Purpose

The color blindness nodes simulates the perception of an input color according to a specific color blindness type.

![Color blindness](/images/color-blindness.png)

## Inputs

| Name        | Type    | Description           | Required |
| :---------- | :------ | :-------------------- | -------- |
| Input color | `Color` | The color to simulate | True     |

## Properties

| Name       | Type     | Description            | Required |
| :--------- | :------- | :--------------------- | -------- |
| Affliction | `String` | A color blindness type | True     |

## Output

| Name   | Type    | Description         |
| :----- | :------ | :------------------ |
| Output | `Color` | The perceived color |
