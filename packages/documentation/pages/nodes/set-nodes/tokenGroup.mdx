# Purpose

The Token Group node allows you to combine multiple tokens and give them metadata such as a name (or in the future description and additional metadata such as type).

For now, all it does is prepend the `name` to every token.

## Inputs

| Name   | Type            | Description                         | Required |
| :----- | :-------------- | :---------------------------------- | -------- |
| Tokens | `SingleToken[]` | The tokens the group should include | True     |

## Properties

| Name | Type     | Description            | Required |
| :--- | :------- | :--------------------- | -------- |
| Name | `String` | The name for the group | True     |

## Output

| Name   | Type            | Description         |
| :----- | :-------------- | :------------------ |
| Output | `SingleToken[]` | The new token group |
