# Purpose

The `And` node provides the ability to perform the boolean [And operation](https://en.wikipedia.org/wiki/Logical_conjunction)

## Inputs

The And node has dynamic inputs, so you can provide 0 - n inputs. Any values provided will be cast to a boolean value and evaluated together in ordered manner

## Output

| Name   | Type      | Description                     |
| :----- | :-------- | :------------------------------ |
| Output | `Boolean` | The result of the And operation |
