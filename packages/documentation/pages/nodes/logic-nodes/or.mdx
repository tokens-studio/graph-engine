# Purpose

The `OR` node provides the ability to perform the boolean [OR operation](https://en.wikipedia.org/wiki/Logical_disjunction)

## Inputs

The Or node has dynamic inputs, so you can provide 0 - n inputs. Any values provided will be cast to a boolean value and evaluated together in ordered manner

## Output

| Name   | Type      | Description                    |
| :----- | :-------- | :----------------------------- |
| Output | `Boolean` | The result of the Or operation |
