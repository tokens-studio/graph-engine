# Purpose

The `absolute` node provides the ability to perform the [mathematical absolute operation](https://en.wikipedia.org/wiki/Absolute_value)

## Inputs

| Name  | Type     | Description                                     |
| :---- | :------- | :---------------------------------------------- |
| Input | `Number` | A numeric value. It can be negative or positive |

## Output

| Name   | Type     | Description                    |
| :----- | :------- | :----------------------------- |
| Output | `Number` | The positive form of the input |
